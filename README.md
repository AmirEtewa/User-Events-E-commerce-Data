# E-commerce Funnel Analysis

## Overview
SQL-based funnel analysis of user events data using Google BigQuery.

## Tools
- Google BigQuery (SQL)
- Looker Studio (visualization)

## Analysis Includes
- Funnel stage drop-off
- Conversion rates by traffic source
- Time to conversion
- Revenue metrics and KPIs

## Dataset
Synthetic e-commerce user events dataset covering December 2025.
